# Объявление переменных: `var`, `let` и `const`

`var`, `let` и `const` — это ключевые слова для объявления переменных в JavaScript.

- `var` имеет функциональную область видимости и подвержен переопределению.
- `let` и `const` имеют блочную область видимости.
- Переменные, объявленные с `let`, можно переназначить.
- `const` нельзя переназначить, хотя сам объект, на который указывает переменная `const`, можно изменить.

---

## `var`

- **Область видимости:** Функциональная (переменная доступна внутри функции, где она объявлена).
- **Переопределение:** Можно повторно объявить ту же переменную.
- **Поднятие (hoisting):** Объявления поднимаются в начало области видимости, но не инициализируются.

### Примеры

```javascript
// Пример 1: var
function showVar() {
  console.log(x); // undefined
  var x = 10;
  console.log(x); // 10
}
showVar();

// Пример 2: var
var a = 5;
var a = 10; // Повторное объявление допустимо
console.log(a); // 10
```

---

## `let`

- **Область видимости:** Блочная (переменная доступна внутри блока `{}`).
- **Переопределение:** Нельзя повторно объявить переменную в одном блоке.
- **Поднятие:** Поднимается в начало блока, но не инициализируется (при попытке доступа выдается ошибка).

### Примеры

```javascript
// Пример 1: let
function showLet() {
  // console.log(y); // ReferenceError: Cannot access 'y' before initialization
  let y = 20;
  console.log(y); // 20
}
showLet();

// Пример 2: let
let b = 15;
// let b = 25; // SyntaxError: Identifier 'b' has already been declared
console.log(b);
```

---

## `const`

- **Область видимости:** Блочная.
- **Переопределение:** Нельзя повторно объявить переменную.
- **Переназначение:** Нельзя переназначить значение переменной.
- **Поднятие:** Поднимается в начало блока, но не инициализируется.

### Примеры

```javascript
// Пример 1: const
const c = 30;
// c = 40; // TypeError: Assignment to constant variable.

// Пример 2: const
const myObject = { key: "value" };
myObject.key = "newValue"; // Можно изменить значение свойства объекта
console.log(myObject); // { key: "newValue" }

// Пример 3: const
// const myObject = { other: "value" }; // SyntaxError: Identifier 'myObject' has already been declared
```
